# Introduction

## What is Fitness Warrior?

Fitness Warrior is a fitness application developed by a team of university students as part of a software development project. It's built using Python for the backend and Flutter for the frontend, connected via a shared database.

## Who should use this app?

Fitness Warrior is mainly aimed at people who are just starting out and beginning their fitness journey, however anyone can use it. The app is for those who want to:

- Start or maintain a fitness routine
- Track their progress over time
- Stay motivated using an XP and level based system
- Get guidance on exercises and nutrition

## Key Features

- Personalised workout plans based on fitness goals
- Progress tracking with visual graphs
- Custom fitness tasks and streaks
- Meal planning complete with nutrtional information
- Controllable fitness notifications
- Workout Guidance videos for each exercise
