# Features

## Account & Registration

Users can create an account by providing their first and last name, email address, 
a secure password, date of birth and a username. The system 
validates all inputs before allowing registration.

Users can log in from any device using their email and password, with account data 
synchronised across all devices.

## Personalised Fitness Plan

Users input their fitness data including weight, height, activity level and diet. 
They then select a personal fitness goal such as losing weight, gaining muscle, or 
maintaining a healthy lifestyle. The system generates a customised fitness plan based 
on this information.

## Notifications

Users receive fitness related notifications including workout reminders, streak updates 
and meal suggestions. Users have full control over their notifications and can enable 
or disable specific notification types at any time.

## Navigation

The app provides a clear navigation system with well organised menus and labelled 
buttons, allowing users to reach any main feature within a small number of steps.

## Custom Tasks

Users can create their own custom fitness tasks linked to their personal goals. Tasks 
can be edited at any time and marked as complete. The system records completed tasks 
and uses this data for progress tracking, streak calculation and reward allocation.

## Rewards System

Users are automatically rewarded when they complete a task. Rewards come in the form 
of avatar items, accessories, cosmetic upgrades and in-game currency. The type and 
rarity of the reward is determined by the difficulty of the completed task. Rewards 
are added to the user's inventory immediately and cannot be earned twice for the same task.

## Progress Tracking

The system stores user fitness data including body weight, workout performance, calories, 
steps, lap times and exercise sets. A line graph displays changes in fitness data over 
time, automatically updating as new data is added.

## Workout Guidance

Users can search for any exercise and the system will display a video tutorial for that 
exercise. Video sources are stored in the database and played directly within the app.

## Meal Collection

Users can browse a collection of meals and view nutritional information including calories 
and protein content. Full ingredients and preparation steps are provided for each meal. 
Meals can be filtered based on dietary needs.

## Workout Plans

The system provides multiple workout plans with multiple difficulty levels. Users can 
choose whether the difficulty increases over time using a slider. Workout plan 
recommendations are tailored to the user's fitness level.
