# Usage Guide

## Overview

This is a walkthrough of how a user could use our app Fitness Warrior.

## 1. Create an Account

Upon opening Fitness Warrior you will be prompted to create an account.

- Enter username and email address
- Create a password
- Click sign up

The account will be created and you will be signed in.

## 2. Completing the Questionnaire

After registering, you will recieve a fitness questionnaire, this information is designed to help you generate a personalised fitness plan based on your goals.

## 3. Browsing Exercises

The app contains a full exercise library that a user can browse and search, the user can also create custom exercises if they wish to do so.

- Use the search bar to find exercises by name
- Filter exercises by body area or equipment
- Select an exercise to view a description and a visual representation of which muscles it is targeting as well as guided video tutorial for that exercise.

## 4. Logging a Workout

To log a completed workout:

1. Select the exercise completed
2. Add the sets and reps acheived
3. Add any specific notes
4. Click save workout

## 5. Workout History

A workout history will be attatched to your profile so as a user you will be able to view past workouts including the excercises, weight, set and rep range.

## 6. Meal Plan

Fitness Warrior includes a meal plan feature:

1. Select which food or recipes you have or will be having that day
2. Adjust the quantity, and the nutritional information will be adjusted accordingly 
3. Your meal plan is saved.

## 7. Tracking Streaks

Fitness Warrior has a streak feature so if a user logs a workout consecutively the workout streak is increased. 

## 8. XP and Levels

Fitness Warrior includes an XP level based system, where you will earn XP for completing workouts and custom tasks. Current XP and level data are displayed on the home page.

## 9. Update your Profile

Details given when first signing up can be changed at any time by going to the profile section of the app and updating.
